Un outil léger et performant qui permet d'extraire et de synthétiser les données d'activité à partir de fichiers CSV. Ce programme automatise la lecture des données brutes, filtre les informations importantes et offre un aperçu global et clair des activités de l'utilisateur ou du système.

### Fonctionnalités principales

- Extraction intelligente : Lit et analyse (parse) les données du fichier CSV, en filtrant les enregistrements d'activité selon des critères spécifiques (temps, type d'activité, utilisateur...).

- Synthèse des données : Regroupe et calcule des statistiques (par exemple : durée totale, fréquence d'apparition de chaque type d'activité).
